## k2-portal 是什么

具有`k2`业务属性的微前端开发框架，其业务涵盖统一登录认证、接口请求、graphql、运行时环境变量(nacos)。

![](../assets/guide_1.png)

## k2-portal 做了什么

竭尽可能让程序员从繁琐的业务设置中解脱出来，专注功能开发。

- 命令行生成项目骨架
- 认证登录登出
- 运行时环境变量
- 统一接口请求入口
- 主题风格定制和运行时切换
- 拆解分割大型应用
- 各应用独立运行与被`Portal`接管运行
- 应用间通信
